## chrome.scripting 脚本

### 动态注入各类代码

```js
// 动态执行JS代码
chrome.scripting.executeScript(
  {
    files: 'JS 文件路径',
    func: '函数名',
    target: {
      tabId: '标签页id',
    }
  },
  callback
)
// 动态执行CSS代码
chrome.scripting.insertCSS(
  injection: {
    css: '样式字符串',
    files: '样式文件路径',
  },
  callback?: function,
)
```

## chrome.cookies

### 获取该站点所有 cookie

```js
const url = 'https://www.baidu.com';
chrome.cookies.getAll({url}, cookies => {};
```

### 删除站点某个 cookie

```js
const url = "https://www.baidu.com";
const cookieName = "userName";
chrome.cookies.remove({ url, name: cookieName }, (details) => {});
```
